Sprint 1 update:

For this sprint, we completed the following activities:

  1. Set up Github organization: https://github.com/orgs/dry-blessing/
  2. Granted permissions to all our team members to access the Github account
  3. Created account on waffle.io which is connected to our Github organizational account
  4. Created Cloud 9 workspaces to work on project code
  5. Created a Heroku application “mgt-656-dry-blessing” at the following link: http://mgt-656-dry-blessing.herokuapp.com/
  6. Organized user stories into the product backlog
  7. Estimated the size of each user story keeping user story 1 as the base
  8. Decided which user stories will be completed in sprint 2
  9. Decided which team member will be responsible for the activities in sprint 2

 

Next steps:

  1. We will be running a sprint planning meeting every Friday morning to estimate the stories and choosing the stories to complete in the coming sprints.
  2. We will work individually and in the stories we find harder to do, we will team up to figure out better solutions faster.
  3. We will get our sprint reports to look nicer!
