Quick Evently Snapshot:

Our team is creating a website that is an affordable alternative to Eventbrite. The site allows users to create new events and post them to the site. It also allows users to view upcoming events and RSVP to those that interest them, provided they have a Yale e-mail address. We launched our website on Yale’s campus and different Yale schools are using it. We analyzed the traffic through google analytics and are now conducting A/B test to improve certain features.

Project Maintenance:

  • Continue checking Google Analytics to understand audience, referral, page visits, and more.
  • Improve graphics on all pages to increase number of views
  • Keep populating new events on the events page and make sure that the browse events page/ events listing on landing page is never empty
  • Make sure that all changes to the website are pushed by one person to herokuapp
  • Keep running gulp in a separate terminal on cloud9 to test changes before pushing to heroku
  • Pay cloud9/ heroku fee (if necessary) to keep the app running!

New Features to Consider Building into the Site:

  • Build separate pages that a) give user detailed contact information of team, b) lets them learn more about the website and its features, c) search for events by city and date, d) categorize events into different buckets by event type, e) give users more information on pricing, security etc.
  • Let organizer of each event put in his own contact details and bio
  • Let people add tags to events
  • Incorporate google maps to show users event locations
  • Build in a payments and donation feature as well
  • Get sponsors on the website!

Features Testing


A/B Test for whether including the text “RSVP” on homepage increases click-through rates to Event Detail pages

What this means: Including “RSVP” next events on the homepage appears to be increasing the conversion rate to the event detail pages, however these are only preliminary results. The “RSVP” variation is showing a 10.4% conversion rate compared to 8.6% without. The development team recommends the clients check the experiment again on Google Analytics in another week or two to discover the final conclusion.

A/B Test for whether “Donate Here” or “Support this event” on event details pages increases click-through rates to Donation page

What this means: “Donate Here” is showing more promise than “Support this event”, however with so few session and conversions it is too soon to reach a solid conclusion. The development team recommends the clients check the experiment again on Google Analytics in another week or two to discover the final conclusion.
